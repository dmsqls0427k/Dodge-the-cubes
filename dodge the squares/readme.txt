EunBin Lee’s READ ME
The game rule is simple. Using the pink block, catch five yellow blocks while avoiding the red blocks! If you win, the screen will turn green, and if you lose the game screen will turn red. 

START button will allow you to start the game.

PAUSE button lets you pause the game and enter a grey screen. Push pause again to re-enter back to the game. If you want to restart, you can push start button while in pause.

LEFT button let’s you move the pink block left.

RIGHT button let’s you move the pink block right.

After winning/ losing, you can restart the game by pressing START button.


Have fun! 0_0
